## Sending logs
- Debug Log
```cs
log.Debug("Hello from log, this is debug");
```

- Info Log
```cs
log.Info("Hello from log, this is info");
```

- Warning Log
```cs
log.Warning("Hello from log, this is warning");
```

- Error Log
```cs
log.Error("Hello from log, this is error");
```
